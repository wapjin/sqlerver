修改数据


路径：
/up_table
POST
参数
table = 数据表名（sql_token）
token = 识别码（eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJleHAiOjE2MjU0MTI4NDMsImlhdCI6MTYyNTMyNjQ0M30.cF0knkos36HfmFfBMBEwp05P9HXrwqZ-vZZJR8Q2YDA）

}
